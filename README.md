# Landing Launchpad

Static dark UI landing template (Webflow-style)

Description
- A lightweight, static landing page built with plain HTML and CSS. Ready for quick deployment to GitHub Pages, Netlify, or Vercel.

Features
- Single-page static site: `index.html` + `css/webflow.css`.
- Assets in the `assets/` directory (images, icons).
- Minimal dependencies — no build step required.

Repository structure
- `index.html` — main entry
- `css/webflow.css` — styles
- `assets/` — images and icons
